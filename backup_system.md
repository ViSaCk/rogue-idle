## 🌙 N 시스템 (루나 / 증분)

### 재화 구조 (최신)

```
마나 (Mana)
  - N의 핵심 숫자. 화면에 크게 표시.
  - 1서클 이해도 = 마나 생산속도 (직결)
  - 마나캡 존재 (매우 큰 수) → 단계 전환 분기점
  - 2단계부터 결계력 = 마나 (침공 방어)
  - 환생 시 리셋

깨달음 (Insight)
  - 환생 재화 (구 파편 역할로 격상)
  - 마인드맵 노드 해금에 사용
  - 환생할 때마다 획득. 누적. 리셋 안 됨.
  - 획득량 = floor(log10(마나 + 1))
  - 킨츠기 테마: "깨진 것들이 쌓여 새로운 길이 된다"

이해도 (Comprehension) ← 신규
  - 각 서클마다 별도 보유. 한계 없음.
  - 집중력 N 투자 → 이해도 N/s 증가
  - 상위 서클 이해도가 하위 서클 이해도 증가속도에 기여
  - 1서클 이해도 자체 = 마나 생산속도
  - 이해도 일정 수치 돌파 시 → 다음 서클 해금
  - 정확한 수치는 밸런스 백업서 참고
```

### 이해도 시스템 (핵심!)

```
안티매터의 차원 보유량과 동일 개념, 용어만 다름

예시:
  t=0: 1서클 집중3, 2서클 집중1
       1서클 이해도 증가속도 = 3(집중) + 1(2서클기여) = 4/s

  t=5: 1서클 이해도 = 20
       1서클 이해도 증가속도 = 3 + 1 + 20 = 24/s
       마나 생산속도 = 24/s

  t=10: 1서클 이해도 = 20 + (24×5) = 140
        마나 생산속도 = 144/s

→ 기하급수적 증가. 마나캡으로 제어 필요.
→ 수식 설계는 밸런스 담당.
```

### 마법서 시스템 (서클)

```
1~8서클 마법서

작동 원리:
  상위 서클 이해도 → 하위 서클 이해도 증가속도에 기여
  1서클 이해도 = 마나 생산속도

집중력 분배:
  - 한정 자원을 각 서클에 배분 (뺄 수 있음)
  - 10단위마다 ×2 보너스

서클 공개 범위:
  - 현재 해금 서클 + 바로 다음 1개만 표시
  - 그 이후는 숨김 (스포일러 방지)

서클 해금 조건:
  - 해당 서클 이해도가 일정 수치 돌파 시
  - 정확한 수치는 밸런스 백업서 참고
```

### 집중력 (Focus)

```
시작: 5
한계: 없음 (계속 늘어남)
역할: 서클에 배분하는 한정 자원

획득처 (50:50 분배):

  [마나 소모 — 초반부터 가능] ← 핵심!
  - 마나를 일정량 소모 → 집중력 +1 구매
  - 비용은 살수록 지수적 증가
  - 처음엔 직접 클릭 (A방식)
  - 루나의 루틴 노드 해금 후 자동화 (C방식)
  - A → C 순서로 진행

  [마인드맵 노드 — 첫 환생 이후]
  - 특정 노드 해금 시 집중력 +N
  - 전략적 선택으로 획득

스토리:
  "루나는 마나를 쏟아부어
   더 넓은 집중의 경지에 닿았다."

  루틴 노드 해금 후:
  "이제 루나의 손은
   더 이상 그 일을 기억하지 않아도 됐다."
```

### 환생 — 마나 회로 재설계

```
개념: 강제 리셋이 아닌 "더 나은 방식으로 다시 짜기"
트리거: 자발적 (1단계) / 침공 실패 시 강제 (2단계~)

리셋 항목:
  - 마나
  - 이해도 (모든 서클)
  - 서클 분배 (집중력 위치)
  - 서클 해금 상태

유지 항목:
  - 깨달음 (누적, 마인드맵용)
  - 마인드맵 해금 상태
  - 집중력 총량
  - 작업대 해금 상태

빨라지는 이유:
  - 집중력 총량 증가
  - 마인드맵 노드 효과 적용됨
  - 플레이어가 최적 분배를 앎
```

### UI 구조

```
메인 화면:
  - 루나의 서재 / 제레마이어의 탑 탭
  - 단계 표시 없음 (플레이어에게 숨김)
    → 직감적으로 느끼게. "내가 1단계구나" 보여주지 않음.

사이드바:
  - 재화 (마나, 집중력, 이해도, 깨달음)
  - 아이디어 지도 슬롯 (첫 환생 후 활성화)
  - 일기

제거된 것:
  - 활성 효과 패널 → 마인드맵 내부로 이동
  - 단계 배지
```

### 마나캡 (Mana Cap)

```
마나에 상한선 존재 (매우 큰 수)
안티매터 1e308과 유사한 개념

역할:
  - 단순 숫자 증분의 최종 목표
  - 1단계에서는 절대 도달 불가
  - 2단계 중반~후반에 처음 도달
  - 캡 돌파 → 단계 전환 분기점 or 주요 계단 장애물
  - 3단계 진입 조건으로 활용 예정

정확한 수치: 밸런스 담당
```

### 루나의 세 트리

```
1. 루나의 아이디어 (마인드맵)
   - N을 강화하는 영구 트리
   - 재화: 깨달음 (환생 재화)
   - 형태: 마인드맵 (중앙에서 퍼져나감)
   - 공개: 바로 다음 노드만 보임, 그 다음은 ???
   - 첫 환생 후 활성화
   - 카테고리:
     · 마나의 흐름 (N 강화)
     · 결계의 술 (방어 강화)
     · 환생의 비밀 (환생 강화)
     · 루나의 루틴 (자동화/QoL)
   - 활성 효과 패널은 마인드맵 내부에 표시

2. 루나의 작업대
   - R을 강화하는 영구 트리
   - 재화: 깨달음 + 일부 R 재료
   - 형태: 격자 (전체 보임, 재화 부족으로 잠김)
   - 해금 시 스토리 메시지 동반
   - 브랜치: 방어구 / 무기 / 부적

3. 루나의 루틴 (마인드맵 내 카테고리)
   - 자동화/QoL 트리
   - 작은 QoL부터 시작
   - 집중력 자동 구매 노드 포함 (A→C 자동화)
   - 최상단: 8서클 완전 자동화 (2단계 전환 조건)
```

### 침공 시스템

```
1단계: 없음
2단계~: 고정값으로 등장

판정: 현재 마나 >= 침공 강도 → 성공
      현재 마나 < 침공 강도  → 강제 환생

강도: 고정값, ×10배씩 증가 (수치는 밸런스 백업서)
```

## ⚔️ R 시스템 (제레마이어 / 로그라이크)

### 핵심 원칙

- **완전 자동전투** (절대 바꾸지 말 것)
- 수치 강화의 카타르시스가 메인
- 유물은 수치를 비대칭하게 만드는 도구
- 스토리는 감초

### 스탯 시스템

```
기본 (처음부터):
  ATK (공격력)
  HP (체력)
  SPD (공격속도, ms 단위 — 클수록 느림)
  시작 공속: 3000ms (뽕맛을 위해 느리게 시작)

중급 (마일스톤 해금):
  DEF (방어력)
  PEN (관통)
  CRIT (치명타 확률)
  CRIT.D (치명타 배율)

고급 (후반):
  EVA (회피)
  PARRY (패리)
  LIFE.S (흡혈)
  RFLCT (반사)

스탯 해금 방식:
  마일스톤 기반 (특정 사건 시)
  루나의 작업대에서 도구 제작으로도 해금
  해금 시 스토리 메시지 동반
```

### 적 시스템

```
5종 시작:
  고블린:    저HP 고SPD 저ATK / 약점: CRIT
  오크 전사: 중HP 저SPD 고ATK / 약점: PEN
  트롤:      고HP 저SPD 자동회복 / 약점: LIFE.S
  도적:      저HP 고SPD 회피 / 약점: 일격
  마법사:    저HP 중SPD 관통 / 강점: 흡혈 무효

변종 (10~15% 확률):
  오크(광폭화): SPD 2배, HP 50%
  고블린(정예): 모든 스탯 1.5배
  트롤(저주):   자동회복 없음, ATK 2배
```

### 맵 구조

```
5층마다 A/B/C 루트 선택:
  A: 전투2 / 이벤트1 / 보물1 / 휴식1
  B: 전투1 / 이벤트3 / 보물1
  C: 전투1 / 이벤트1 / 엘리트1 / 휴식2

이벤트: 즉시효과 70% + 텍스트선택 30%
보스층: 5의 배수
```

### 유물 시스템

```
지연 선택 + 자동 안전망:
  런 중 획득 → 가방 보관
  휴식 칸에서 정리
  일반 유물: 미선택 시 자동 적용
  희귀/전설: 직접 선택 필수

등급:
  🟢 일반: 기본 스탯 +% (자동 적용 가능)
  🔵 희귀: 조건부 효과 (직접 선택)
  🟡 전설: 런 변형 (직접 선택, 희귀)

원칙: 고급 스탯은 유물에 잘 안 넣음 (밸런스 보호)
```

### 루나의 작업대 (R 메타 진행)

```
루나 시점의 격자형 스킬트리
재화: 깨달음 + R 재료
해금 시 스토리 메시지 동반
부녀 협동의 시스템적 표현

브랜치: 방어구 / 무기 / 부적
```

## 🔗 R ↔ N 연결고리

```
N → R:
  마력 → 전사 마법 버프
  작업대 노드 → 스탯/능력 해금

R → N:
  5층마다 → 깨달음 보너스
  특수 재료 → 마인드맵/작업대 특수 노드 해금
  탑에서 책 → 깨달음 이중 경로 B

두 모드의 동등함:
  N만 → 정체 (한계 도달)
  R만 → 마나 부족, 강화 못함
  둘 다 → 시너지 폭발
```
