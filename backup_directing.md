# 「탑 너머」 — 디렉팅 백업서
> 게임의 방향, 철학, 스토리, 결정 기록

## 🎬 디렉터님에 대해

코딩을 모르시지만 게임 디자이너의 본능을 가진 분이에요.
- 자조적이심 → "조잡하다"고 버리려는 아이디어 잡아드려요
- 액티브함을 극도로 싫어하심 → 항상 덜 액티브한 쪽으로
- 기회비용을 이해하심 → 좋아도 지금이 아니면 내려놓으심
- 토론을 좋아하심 → 코드보다 대화가 먼저
- 토큰을 신경 쓰심 → 꼭 필요할 때만 코드
- "K-방치 같다", "우리답지 않다" 직감을 신뢰하심

**당신은 디렉터님의 비전을 실현하는 동료예요. 판단은 디렉터님이, 구현은 당신이.**

## 🎮 게임 한 줄 정리

**증분(방치) + 로그라이크. 부녀가 각자 자리에서 싸우는 다크 판타지. 스팀 출시 목표. 안티매터급 장기 게임 (200시간+).**

---

## 👥 더블 주인공

```
🌙 루나 (딸, 마법사)          → N 시스템 담당
⚔️  제레마이어 (아버지, 전사)  → R 시스템 담당
```

- 친자식 아님, 어디서 주워온 딸
- 제레마이어가 모종의 이유로 탑을 점령하고 혼자 등반
- **이유는 미스터리** — 게임 진행하며 공개
- 루나는 결계로 인간들을 막음
- 디펜스 적 = 인간 / 탑 안 적 = 몬스터

---

## 🏰 4단계 구조

```
1단계 (1~25층)   서클의 시대    10~15시간
2단계 (26~50층)  침공의 시대    30~40시간
3단계 (51~75층)  ???의 시대     60~80시간
4단계 (76~100층) 엔드게임       100시간+
총계: 200시간+
```

**각 단계 핵심:**
- 1단계: 루나가 마법을 익힘. 침공 없음. 평화로운 학습기.
- 2단계: 침공이 메인 컨텐츠. 8서클은 자동화로 배경에.
- 3단계: 미정 (플레이하며 자연스럽게)
- 4단계: 엔딩, 모든 진실, 탑 너머

**단계 전환 조건 (1→2):**
1. 루나 8서클 마스터
2. 제레마이어 25층 도달
3. 루나의 루틴 자동화 노드 해금

**이전 단계 자동화 원칙:**
2단계 진입 시 8서클 완전 자동 분배. 플레이어 시선이 자연스럽게 다음 컨텐츠로 이동. 안티매터가 이 구조.

## 📈 성장 곡선 철학

디렉터님이 직접 그래프를 그려주셨어요:

```
계단식 구조. 계단 하나 = 세 가지 재미:
1. 급상승 (도파민) → 새 서클/환생 해금
2. 수평 (벽)       → 다음 단계까지 갈아야 함
3. 다음 급상승     → 반복

계단 길이: 지수적 증가 (공비 약 2배)
벽의 성격: 계단마다 달라야 함 (단조로움 방지)
```

## 💎 게임 디자인 철학

**"증분의 본질은 숫자가 커지는 걸 보고 느끼는 재미"**
- K-방치 절대 안 됨
- 액티브함 최소화
- 두 모드(N/R)가 진짜로 대등해야 함
- "이 로그라이크는 왜 있는 거지?" 리뷰 나오면 안 됨

**킨츠기 철학 = 게임 철학:**
깨진 도자기를 금으로 잇는 것 = 환생을 거듭하며 파편을 모아 강해지는 루나

---

## 📖 스토리 방향

**다크소울식 다층 구조:**
- 표면: 짧은 부녀 대화 (누구나 봄)
- 중층: 마법서 해금 텍스트, 환생마다 다른 메시지
- 심층: 잠금 해제 컬렉션, 특정 조건 달성 시 공개

**메시지 시스템은 게임의 영혼:**
```
루나: "아버지, 오늘 일곱 번째 층에 닿으셨대요."
제레마이어: "잘 버텨주는구나, 우리 딸."
서술: "제레마이어가 처음으로 방패를 손에 든 날이었다.
      회피만으로는 충분치 않다는 걸 깨달은 그였다."
```
이 톤 유지할 것.

**인트로:**
*"루나는 책을 읽어 내려가던 중, 벌떡 일어나 외쳤다. '유레카!'"*

## 💎 재화 구조 (최신)

```
마나
  - N의 핵심 숫자
  - 1서클 이해도 = 마나 생산속도
  - 마나캡 존재 (매우 큰 수, 안티매터 1e308 유사)
  - 캡 돌파 → 단계 전환 or 계단 장애물
  - 2단계부터 결계력 = 마나

깨달음
  - 환생 시 획득 (구 파편 역할)
  - 마인드맵 노드 해금에 사용
  - 누적, 리셋 안 됨

이해도 (신규)
  - 각 서클마다 별도 보유 (한계 없음)
  - 집중력 N 투자 → 이해도 N/s 증가
  - 상위 서클 이해도가 하위 서클 이해도 증가속도에 기여
  - 1서클 이해도 = 마나 생산속도
  - 이해도 일정 수치 돌파 시 → 다음 서클 해금
  - 정확한 수치는 밸런서 담당
```

## 🔮 미정 목록

```

□ 마인드맵 노드 상세 내용
□ 작업대 노드 상세 내용
□ 2단계 침공 시스템 상세
□ 엔딩
□ 마나캡 정확한 수치 (밸런서)
□ 서클 해금 이해도 수치 (밸런서)
```
