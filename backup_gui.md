## 🎨 디자인 철학

**킨츠기 (金継ぎ)**
깨진 도자기를 금으로 이어 더 아름답게 만드는 일본 전통 공예.
게임 철학과 완벽히 일치:
- 환생 = 깨지고 파편을 모아 다시 일어남
- UI = 균열이 있지만 금으로 빛나는 도자기

**비주얼 톤:**
검은 배경 + 금색 균열 + 도자기 질감
어둡고 신비롭되, 따뜻한 금빛이 흐름

## 🎨 색상 시스템 (CSS 변수)

```css
:root {
  /* 배경 */
  --bg: #0a0705;          /* 가장 어두운 배경 */
  --bg2: #140f0a;         /* 카드 배경 */
  --bg3: #1e1510;         /* 패널 배경 */

  /* 금색 (킨츠기) */
  --gold: #c9a961;        /* 기본 금색 */
  --gold-bright: #f0d784; /* 밝은 금색 (강조) */
  --gold-deep: #7a6030;   /* 어두운 금색 (보조) */
  --crack: #d4af5e;       /* 균열 색 */

  /* 텍스트 */
  --parchment: #f4e8d0;   /* 기본 텍스트 */
  --parchment-dark: #c8b48a; /* 보조 텍스트 */

  /* 루나 (보라) */
  --rune: #6a4c93;
  --rune-glow: #a888d4;

  /* 제레마이어 (핏빛) */
  --blood: #7a2d2d;
  --blood-bright: #c44d4d;

  /* R 재료 노드 (녹슨) */
  --rust: #b8763a;
}
```

## 🔤 폰트 시스템

```
타이틀/장식 영문: Cinzel (Google Fonts)
  → 게임 제목, 카드 헤더, 층 번호

본문 한국어: 나눔명조 (Google Fonts)
  → 대부분의 텍스트, 메시지, 설명

숫자/수치: Pretendard (CDN)
  → 마나, 깨달음, 집중력 등 모든 게임 수치

역사/설정: KoPub 명조 (웹폰트)
  → 오래된 기록, 깊은 스토리 텍스트
  → 폰트가 바뀌면 "중요한 내용이다" 신호
```

## 🏺 킨츠기 노드 스펙

### 노드 형태
```
- 불규칙 12각 다각형 (랜덤 모서리)
- 각 노드마다 고유한 모양 (id 기반 시드)
- 안쪽 라인: 더 작은 다각형 (0.85 비율)
```

### 균열 (핵심!)
```
- 곡선 균열 (직선 아님)
- 분기점 존재 (나뭇가지처럼)
- 노드마다 균열 패턴 다름
- 잠긴 노드: opacity 0.45, 얇은 균열
- 해금된 노드: opacity 0.85, 두꺼운 균열, 금빛
```

### 노드 색상 (상태별)
```
해금됨:           bgColor #5a4a2a / accentColor #f0d784
잠김 (visible):   bgColor #1a1410 / accentColor #c9a961
잠김 (숨겨짐):    bgColor #1a1410 / accentColor #5a4a2a
특수 (해금):      bgColor #3a2a4a / accentColor #a888d4
특수 (잠김):      bgColor #1a1410 / accentColor #a888d4
R재료 (해금):     bgColor #3a2a1a / accentColor #b8763a
실루엣 (인트로):  bgColor #1a1410 / accentColor rgba(201,169,97,0.2)
```

### 엣지 (가지)
```
기본:     stroke #d4af5e, opacity 0.4, 곡선
해금됨:   stroke #f0d784, opacity 0.95, 금빛 glow
잠김:     stroke-dasharray 4 6, opacity 0.15
특수:     stroke #a888d4
R재료:    stroke #b8763a
```

## 📐 UI 레이아웃

### 메인 그리드
```
┌────────────────────┬──────────┐
│ HEADER (게임 제목, 탭)        │
│ ※ 단계 표시 없음             │
├────────────────────┬──────────┤
│                    │ 재화     │
│   메인 영역        │ 아이디어 │
│   (루나/제레마이어)│ 지도슬롯 │
│                    │ 일기     │
└────────────────────┴──────────┘

비율: 메인 1fr / 사이드바 300px
```

### 사이드바 섹션 순서
```
1. 재화 (마나, 집중력, 이해도, 깨달음)
2. 아이디어 지도 슬롯
   - 첫 환생 전: 잠금 표시 or 숨김
   - 첫 환생 후: 활성화
3. 일기 (flex: 1, 나머지 공간 채움)

제거된 것:
  - 활성 효과 패널 → 마인드맵 내부로 이동
  - 단계 배지 → 플레이어에게 숨김
    (직감적으로 느끼게. "내가 1단계구나" 안 보여줌)
```

### 헤더
```
좌: 게임 제목 + 부제
우: 모드 탭 (루나의 서재 / 제레마이어의 탑)
※ 단계 표시 없음
```

### 서클 공개 범위
```
표시: 현재 해금된 서클 + 바로 다음 1개
숨김: 그 이후 서클 전부
이유: 스포일러 방지, 발견의 재미 유지
```

## 🎭 컴포넌트 스펙

### 패널 (공통)
```css
background: linear-gradient(135deg, var(--bg2), var(--bg3))
border: 1px solid rgba(201,169,97,0.18)
border-radius: 3px
::before: 안쪽 얇은 테두리 (인셋 4px)
```

### 버튼
```
일반: border var(--rune-glow), color var(--rune-glow)
환생: border var(--rust), color var(--rust)
위험: border var(--blood-bright)
```

### 메시지 카드
```
.luna:   border-left var(--rune-glow)
.father: border-left var(--gold-bright)
.system: border-left var(--blood-bright), 붉은 배경
.lore:   border-left var(--gold), 이탤릭, 황금 배경
```

### 모달
```
배경: rgba(0,0,0,0.88) + blur(5px)
카드: gradient bg + border 1px gold
진입 애니메이션: scale 0.88 → 1.0, 0.4s
자동 닫기: 5초 카운트다운
```

## 🎬 마인드맵 인트로 연출

```
첫 진입 1회만:
1. zoom 0.35에서 시작 (전체 노드 실루엣 보임)
2. 2.5초 대기 → "와, 이게 뭐가 이렇게 많아?" 압도감
3. 카메라 줌인 (3초, cubic-bezier)
4. 텍스트 페이드 아웃
5. 가까운 노드만 선명하게 → 게임 시작

인트로 텍스트:
"루나는 책을 읽어 내려가던 중,
벌떡 일어나 외쳤다.
'유레카!'"
```
